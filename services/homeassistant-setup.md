# Home Assistant

**Host:** Proxmox VM/LXC (192.168.0.114)
**Port:** 8123
**URL:** http://192.168.0.114:8123
**Zugriff:** Heimnetz / VPN

## Beschreibung

Home Assistant ist eine Open-Source Smart Home Plattform zur Steuerung und Automatisierung von IoT-Geräten. Die Instanz läuft als VM auf dem Proxmox Host (192.168.0.101).

Aktuell sind keine Geräte angebunden — die Installation ist vorbereitet für zukünftige Nutzung.
